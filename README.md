# 🍽️ Zomato Restaurant Data Analysis

## 📌 Overview
This project performs **Exploratory Data Analysis (EDA)** on the Zomato Restaurant Dataset using Python. The analysis explores restaurant ratings, cuisines, costs, online ordering, and customer preferences through statistical methods and visualizations.

---

## 🎯 Aim
To perform Exploratory Data Analysis (EDA) on the Zomato restaurant dataset and derive meaningful insights about ratings, cuisines, costs, and customer preferences.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed
- Dataset Understanding
- Data Cleaning
- Descriptive Statistics
- City-wise Restaurant Analysis
- Cuisine Analysis
- Rating Analysis
- Outlier Detection (IQR Method)
- Online Delivery Analysis
- Cost Analysis
- Correlation Analysis
- Feature Engineering

---

## 📈 Key Insights
- 📍 Delhi had the highest number of restaurants.
- 🍝 Italian was the most common cuisine.
- ⭐ Fast Food had the highest average rating (4.6).
- 📊 Most restaurant ratings were above 4.0.
- 👍 Restaurants with more customer votes generally received better ratings.
